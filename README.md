# AWS Resource Tracker

A Bash-based AWS monitoring tool that helps track AWS resources and provides basic Free Tier cost warnings using AWS CLI.

## Features

* Track S3 Buckets
* Track EC2 Instances
* Track Lambda Functions
* Track IAM Users
* Track EBS Volumes
* Track VPCs
* Track Security Groups
* Track Elastic IPs
* Generate AWS resource reports
* Detect running EC2 instances
* Basic Free Tier monitoring

## AWS Services Used

* Amazon EC2
* Amazon S3
* AWS Lambda
* AWS IAM
* Amazon EBS
* Amazon VPC

## Prerequisites

* AWS CLI installed
* Configured AWS credentials
* Linux environment

## Installation

```bash
git clone https://github.com/abhaysgitt/aws-resource-tracker.git
cd aws-resource-tracker
chmod +x aws_resource_tracker.sh
```

## Usage

```bash
./aws_resource_tracker.sh
```
